🔤 编解码工具测试

功能说明

  • 支持 Base64、Hex、URL、Unicode 四种编码格式
  • 可进行编码和解码操作
  • 提供快捷操作按钮（复制、交换、清空）
  • 实时结果显示和错误提示

测试用例

Base64 测试

原文: Hello World! 你好世界
Base64编码: SGVsbG8gV29ybGQhIOS9oOWlve+8jOS4lueVjO+8gQ==

Hex 测试

原文: ABC
Hex编码: 414243

URL 测试

原文: hello world?
URL编码: hello%20world%3F

Unicode 测试

原文: 中
Unicode编码: \u4e2d

使用说明

  1. 选择操作类型（编码/解码）
  2. 选择编码方式（Base64/Hex/URL/Unicode）
  3. 在输入框中输入要处理的文本
  4. 点击"编码"或"解码"按钮执行操作
  5. 结果将显示在输出框中
  6. 可使用快捷按钮进行复制、交换等操作
此测试页面用于验证编解码工具的核心功能。
